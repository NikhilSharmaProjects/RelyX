# RelyX Chrome Extension - Installation Guide

## How to Download and Install RelyX

### Step 1: Visit the Chrome Web Store
1. Open your Chrome browser
2. Go to the Chrome Web Store at **chromewebstore.google.com**
3. Search for **"RelyX"** in the search bar

### Step 2: Find RelyX Extension
- Look for the RelyX icon in the search results
- Click on the RelyX extension card to open its details page

### Step 3: Click "Add to Chrome"
1. On the RelyX extension page, click the blue **"Add to Chrome"** button
2. A confirmation dialog will appear

### Step 4: Confirm Installation
1. Click **"Add extension"** in the confirmation dialog
2. Wait for the installation to complete (usually takes a few seconds)

### Step 5: Pin the Extension (Optional)
1. Click the **puzzle icon** (Extensions) in the top-right corner of Chrome
2. Find **RelyX** in the list
3. Click the **pin icon** to add RelyX to your toolbar for easy access

### Step 6: Start Using RelyX
- Once installed, RelyX will automatically protect you when browsing
- Click the RelyX icon to:
  - View your threat analysis reports
  - Check security status of websites
  - Access extension settings

## Troubleshooting

### Extension Not Appearing?
- Click the **Extensions** icon (puzzle piece) in Chrome toolbar
- Search for RelyX in the list
- Click the pin icon to display it

### Not Blocking Threats?
1. Make sure you're on the latest version of Chrome
2. Click **RelyX icon** → **Settings** → Check if protection is enabled
3. Refresh the webpage

### Performance Issues?
- Close unused browser tabs
- Clear Chrome cache regularly
- Disable other heavy extensions if needed

### Need Help?
- Visit our GitHub repository or contact support
